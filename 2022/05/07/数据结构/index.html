线性表

顺序存储（顺序表）

栈（先进后出）

n个不同元素入栈，出栈元素一共有这么多不同的排列方式（Catalan数）：
$$
\frac{1}{n+1} \times C^{n}_{2n}
$$

  • 空栈：top == -1（最上面的索引为-1）

  • 满栈：top == MaxSize - 1（最上面的索引为栈长度-1）

  • 入栈：

    1. top = top + 1（最上面的索引+1）
    2. stack[top] = new（新入栈的元素）

  • 出栈：

    1. temp = stack[top]（temp存储出栈元素）
    2. top = top - 1（索引-1）

  • “上溢”：栈满存入

  • “下溢”：栈空出栈

共享栈（双栈）

  • 空栈：top1 == -1 top2 == MaxSize

  • 满栈：top1 == top2 - 1

  • 栈1入栈：

    1. top1 = top1 + 1
    2. stack[top1] = new

  • 栈1出栈：

    1. temp = stack[top1]
    2. Top1 = top1 - 1

  • 栈2入栈：

    1. top2 = top2 - 1
    2. stack[top2] = new

  • 栈2出栈：

    1. temp = stack[top2]
    2. Top2 = top2 + 1

循环队列

  • 空队：front == rear

  • 满队：(rear + 1) % MaxSize == front

  • 入队：

    1. queue[rear] = new
    2. rear = (rear + 1) % MaxSize

  • 出队：

    1. temp = queue[front]
    2. front = (front + 1) % MaxSize

链式存储（单链表、双链表、单循环链表、双循环链表、静态链表）

链栈

  • 空栈：top == null

  • 满栈：不存在

  • 入栈：

    1. new -> next = top
    2. top = p

  • 出栈：

    1. temp = top -> data
    2. top = top -> next

链队

  • 空队：front == null , rear == null

  • 满队：不存在

  • 入队：

    1. rear -> next = new
    2. rear = rear -> next

  • 出队：

    1. temp = front -> data
    2. front = front -> next
